# NodeEditor — визуальный редактор HTML-форм
---

## 📖 Описание

NodeEditor — это приложение для создания HTML-форм, опросов, тестов и анкет с помощью визуального редактора нодов.

Пользователь строит структуру формы через систему связанных между собой нодов. Каждый нод можно настраивать, добавлять на сцену и соединять с другими, формируя логику будущей формы.

На основе созданной схемы планируется генерация HTML-формы с возможностью сохранения данных:
- на сервере (в будущих версиях)
- локально в формате CSV (в разработке)

---

## ⚙️ Возможности

- Визуальное создание структуры формы
- Работа с нодами и связями между ними
- Настройка параметров элементов
- Удобный графический интерфейс
- Основа для генерации HTML-форм
